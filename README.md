# Student Attendance Management System

This is a **Python project** that manages student attendance using a **singly linked list**. It allows you to store student details, calculate attendance percentages, and identify students whose attendance is below 75%.

## Features
- Add student details (ID, Name, Total Classes, Attended Classes)
- Calculate attendance percentage for each student
- Display all student data
- Display students with attendance shortage (<75%)

## Classes
- `Student`  
  Stores individual student information and calculates attendance.

- `StudentList`  
  Implements a linked list to manage multiple students and provides methods to display data and check attendance shortages.
